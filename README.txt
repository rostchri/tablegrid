= Tablegrid

Generic table-helper for creating table-grid views. The original idea comes from http://coryodaniel.com/index.php/2011/02/16/hamlburgerhelper-sets-the-table-easily-create-and-display-standard-tables-in-rails/
or http://dzone.com/snippets/hamlburger-helper-easily.

Usage: 
1) In Gemfile: gem 'tablegrid', :git => 'https://github.com/rostchri/tablegrid.git'
2) In application.js: //= require tablegrid
3) In application.css: *= require tablegrid

Requirements: Haml
